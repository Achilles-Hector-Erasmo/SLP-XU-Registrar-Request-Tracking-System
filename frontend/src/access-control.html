System Roles

Define and manage roles and their associated permissions.

Role Name Description Actions
System Administrator Manages access control, whitelists users, assigns roles, and has full system control.
University Registrar All CRUD operations, can add intern/student assistant, assign permissions, and manage documents.
Evaluator Create data entry, edit status, mark as "Request Issued" or "Printed", and process documents.
Student Assistant Update/verify request information per registrar guidelines and coordinate with registrar.
Intern Limited access for verification tasks with read-only permissions.
